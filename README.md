# BaniLoad V1

Türkiye'nin lojistik platformu. Firma ve kamyoncuları buluşturan yük ilanı sistemi.

## Özellikler
- Firma paneli (yük ilanı verme, kamyoncu seçme)
- Kamyoncu paneli (boş araç bildirme, ilanları görme, onaylama)
- Admin paneli (kullanıcı onaylama, ödeme yönetimi, raporlar)
- Harita entegrasyonu
- Güvenli ödeme sistemi (İyzico)
- Hukuki sözleşme yönetimi

## Stack
- Backend: NestJS + TypeScript + PostgreSQL
- Frontend: Next.js 14
- Deploy: Railway (backend) + Vercel (frontend)
